Aqui em baixo tem uma imagem do logo do HTML

Para colocar uma imagem é necessário usar a tag <img>

Negrito e itálico

Para usar algo em negrito precisamos usar a tag <strong>

Para usar algo em itálico precisamos da tag <em>

Usando Marca Texto

Você pode marcar um texto que é importante usando a tag <mark>
Ex: Aqui está um texto grifado com marca texto

Diminuindo as letras

Você pode usar a tag <small>
Ex: Aqui está um texto pequeno

Texto deletado

O texto pode até ser lido, mas deve ser desconsiderado pelo leitor. Pode se fazer isso com a tag <del>
Ex: Aqui está um texto deletado

Texto inserido

É o oposto do texto deletado, ele deve ser lido pelo leitor e o leitor deve prestar atenção. Para fazermos isso podemos usar a tag <ins>
Ex: Aqui está um texto que o leitor deve prestar atenção

Texto subescrito e subscrito

Para colocar alguma coisa subescrita, precisamos usar a tag <sub>
Ex:H2O agora subscrito H20 e sobescrito x2 e x2

Citações

Podemos fazer citações com a tag <q>
Ex: Segundo maquiavel é melhor ser temido ou amado

abreviações

podemos usar a tag <abbr> para abreviar alguma coisa.
Ex: Estou estudando HTML

LISTAS

Listar ordenadas

para fazermos uma lista precisamos da tag <ol> e da tag <li> para colocarmos os itens na lista. Essas tags são utilizadas quando a ordem da lista importa.
Ex:

  1. Item1
  2. item2
  3. item3
  4. item4

Ela também pode usar type para definir o tipo de marcação antes do item

Não ordenadas

Essas lisstas são que a ordem não importa.. pata fazer essa lista precisamos da tag <ul>.
Ex:

  • item1
  • item2
  • item3
  • item4
